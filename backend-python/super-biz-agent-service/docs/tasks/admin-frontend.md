# admin-frontend：管理后台前端

目标：提供可迁移到 Vue/React 的前后端分离式管理后台页面，先在当前静态前端体系中落地。当前版本采用单一登录用户权限模型，任意已登录用户都可以进入后台并使用配置、知识库和统计功能。
